## Development

    $ yarn install
    $ yarn run dev

## Credentials

Pass AWS credendials through the environment variables `AWS_ACCESS_KEY_ID` and `AWS_SECRET_ACCESS_KEY`. Edit the database credentials inside `server/config/index.example.js`, then rename the file to `index.js`.
